통합 모니터링 시스템 개발

  • skills. Java / RV(TIBCO Rendezvous) / Spring / Oracle / Linux
  • dev. 2017.05 ~ 2019.07
  • service. SK HYNIX, LG Display

TIBCO Rendezvous 자체 및 Rendezvous를 사용하는 시스템을 모니터링하여 시스템에서 발생 가능성이 존재하는 장애들을 사전에 예방하는 솔루션입니다. TIBCO Rendezvous 통신을 통해 발생한 Queue 및 각 FAB, HOST, APP, PROCESS 정보를 실시간 또는 특정 이력에 대해 모니터링할 수 있습니다. 미들웨어(RVD) 환경에서 각종 상태 및 데이터를 실시간으로 감지·분석하며, 직관적인 모니터링 UI를 제공합니다.

서버와 프로세스 자원을 수집하여 모니터링 서버로 전송하고, Application의 Alive, Active/Standby, Deadlock 상태를 실시간으로 감지하는 Agent 서버 모듈을 개발 하였습니다. 추후 Front-end 개발 작업에도 서포트로 참여하였습니다.

위 프로젝트는 설계(DB, Architecture)부터 배포(Linux Packaing) 단계까지 모두 참여하였습니다.
