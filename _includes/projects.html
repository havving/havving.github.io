실시간 통합 모니터링 시스템 개발

  • skills. Java / RV(TIBCO Rendezvous) / Spring / Oracle / Linux
  • dev. 2017.05 ~ 2019.07
  • service. SK HYNIX, LG Display

TIBCO Rendezvous 자체 및 Rendezvous를 사용하는 시스템을 모니터링하여 발생 가능성이 존재하는 장애들을 사전에 예방하는 시스템입니다. RV 통신을 통해 발생한 Queue 및 각 FAB, HOST, APP, PROCESS 정보를 실시간으로 감지하고 분석하며 특정 이력에 대해 모니터링할 수 있습니다.

본 프로젝트는 설계(DB, Architecture)부터 개발, 테스트, 배포(Packaing), 운영 전 프로세스에 모두 참여했습니다.

  • dev. 2016.12 ~ 2017.04
  • service. SK HYNIX, LG Display

AI 콘크리트 균열 탐지 학습 모델 개발

  • skills. Python / Tensorflow / Deep-Learning / Kubernetes / Docker
  • dev. 2019.08 ~ 2020.10

AI 딥러닝 학습을 통해 콘크리트 구조물의 미세한 균열을 탐지합니다. 먼저 균열 이미지에 대한 이미지 프로세싱을 통해 잡음을 제거하고, 딥러닝 기법 중 CNN(Convolution Nerual Network, 합성곱신경망)을 사용해 데이터 특징들의 패턴을 파악하여 균열 여부를 판단합니다. 그 후, U-Net, DeepLab V3 등의 Semantic Segmentation 특징 추출 모델을 통해 학습하고 분석하여 균열을 추출합니다.

학습 모델 개발 중, 균열 폭이 가늘게 있는 경우(2~3px 이하) 학습이 제대로 수행되지 않는 현상에 대해 오차함수에 가중치를 부여하여 모델 학습이 가능하도록 하는 연구를 진행했습니다. 연구 결과, 정확도 99% 이상, 조화평균 89~92%의 신뢰성 높은 수치를 기록하며 미세 균열에도 학습을 성공적으로 해내어 추후 특허 출원 및 논문으로도 작성되었습니다.

네이버 클라우드 플랫폼의 클라우드 서비스 중 하나인 가상 GPU 병렬 컴퓨팅을 이용하여 동시에 여러 대의 서버에서 고성능의 연산 처리를 진행했습니다. 또한 Kubernetes를 통해 개발에 필요한 기능들(Tensorflow, Jupyter Notebook 등)을 Docker로 작업하며 운영하기도 했습니다.
